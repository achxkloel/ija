# IJA, UML Editor

## Authors
- Evgeny Torbin <xtorbi00@vut.cz>
- Lukáš Vincenc <xvince01@vut.cz>

## Description
Aplikace pro zobrazení a editaci třídních/sekvenčních diagramů.

## Spuštění pomoci Maven

```
mvn clean compile -PoutputDir javadoc:javadoc assembly:single
```

Spustitelný soubor `ija-app.jar` bude vygenerován do složky `dest`.
